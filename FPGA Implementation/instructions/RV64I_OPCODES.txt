RV64I (+Zicsr/+Zifencei + M-mode SYSTEM used by this core) — Opcode Reference
=============================================================================

Notes
-----
- This core implements: RV64I + RV64M + Zicsr + Zifencei + RV64 “W” subset.
- Instruction length: **32-bit only** (no compressed C extension in this repo).
- Fields:
  - opcode: instr[6:0]
  - funct3: instr[14:12]
  - funct7: instr[31:25] (R-type / shifts / M-extension)
- Hex shown for opcode is the 7-bit value placed in bits [6:0].

Legend
------
R-type:  funct7 | rs2 | rs1 | funct3 | rd | opcode
I-type:  imm[11:0] | rs1 | funct3 | rd | opcode
S-type:  imm[11:5] | rs2 | rs1 | funct3 | imm[4:0] | opcode
B-type:  imm[12|10:5] | rs2 | rs1 | funct3 | imm[4:1|11] | opcode
U-type:  imm[31:12] | rd | opcode
J-type:  imm[20|10:1|11|19:12] | rd | opcode

-------------------------------------------------------------------------------

-------------------------------------------------------------------------------
LUI      rd, imm20      opcode=0x37 (0110111)   U-type  x[rd] = sext(imm20<<12)
AUIPC    rd, imm20      opcode=0x17 (0010111)   U-type  x[rd] = pc + sext(imm20<<12)
